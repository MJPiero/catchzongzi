#CatchZongzi

这是一个端午节接粽子的小游戏，在移动端运行，微信上读取用户信息并记录游戏结果。游戏结束分数提交显示排行榜。

基于[CreateJS](http://www.createjs.com/)引擎开发。

CatchZongzi is a simple game on web that was made by createjs.
